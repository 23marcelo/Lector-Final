Para instalar el driver lector se requieren paquetes adicionales:
- Kernel-devel
- El compilador gcc


Tras una instalacion exitosa se muestra una pantalla similar a la que se muestra en "install.jpg"

Finalmente se requiere modificar el archivo ubicado en "", en dicho archivo se cambia el valor de la variable "" a valor cero, el archivo debera quedar como se muestra en "archivo.jpg"

Para realizar la modificacion del archivo puede hacerlo utilizando el editor de textos que tenga con uno de estos comandos:
	sudo gedit ...
	sudo pluma ...
